## Rodando o projeto

- Instalando dependências

```bash
npm install
```

- Startando

```bash
npm start
```

## Bibliotecas

- `react-router-dom`
- `react-toastify`
- `axios`
- `netlify-cli`

## Deploy

- Build

```bash
npm run build
```

- Instalando cli do `Netlify`

```bash
npm install -g netlify-cli
```
- Deploy via cli

```bash
netlify deploy
```
- Publish directory: `./build`
